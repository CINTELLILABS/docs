### Clone phone number settings on purchase

You can now duplicate the configuration of an existing inbound phone number when purchasing a new one, so you no longer have to rebuild the setup field-by-field for each line.

- Open the Phone Numbers dashboard and select **Clone** next to any inbound number you own to start a new purchase pre-linked to that source
- The purchase screen shows a banner confirming which number is being cloned and which settings will carry over — pathway, variables, voice, webhook, and advanced inbound config
- After the new number is purchased, the cloned settings are applied automatically and you're taken straight to the new number's Call tab so you can review and adjust before saving
- Every cloned field remains editable on the new number, so you can fork a working setup and tweak only what needs to change for the new line

<Note>
  Cloning is available for inbound numbers purchased through Bland. Numbers brought via BYOT (custom Twilio) and SIP numbers don't show the Clone option — port or configure those through their existing flows.
</Note>

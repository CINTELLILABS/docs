### Improvements

**Call Logs & Management**
- [Enterprise] Guard rail triggers for calls are now surfaced directly in call logs
- Added search bar for pathway filtering in call logs
- Added pathway version information to call logs and [get call details API](/api-v1/get/calls-id) responses
- Recording speed selection no longer gets reset when pausing call audio

**Pathways & Routing**
- Added Cmd+K keyboard shortcut to open node search in pathway editor, with Esc to close
- Include SMS send status in pathway logs when using standalone SMS nodes
- Resolved pathway configuration version that was invisibly overriding pathway version in test chat modal

**Send Call Configuration**
- Set answered_by parameter to default true in pathway send call UI
- After editing a call configuration, you can now save it as an entirely new configuration

**Knowledge Base**
- Resolved UI bug when editing knowledge base name and description
- Added clarity that users can exit knowledge base website scraping while it continues running in the background

**Personas**
- Stopped personas condition text boxes from closing prematurely during editing

### Noise cancellation on by default

Noise cancellation now defaults to `true` for all outbound calls, inbound numbers, and pathways. New calls placed without explicitly setting `noise_cancellation` will have background noise filtered from the audio stream automatically.

- The default for `noise_cancellation` on [`POST /v1/calls`](/api-v1/post/calls) and [`POST /v1/inbound/{phone_number}`](/api-v1/post/inbound-number-update) is now `true`
- The same default applies to pathway calls and existing inbound numbers that have never had the field set
- To keep the previous behavior, pass `"noise_cancellation": false` explicitly when sending a call or updating an inbound number

```json
{
  "phone_number": "+15551234567",
  "task": "...",
  "noise_cancellation": false
}
```

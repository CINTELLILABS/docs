This quickstart walks you through sending your first AI phone call with the Bland API. By the end, you'll have placed a live call from a script.

## Prerequisites

- A [Bland account](https://app.bland.ai)
- Your API key from the [dashboard](https://app.bland.ai/dashboard/settings)
- A phone number to call (include the country code, e.g. `+15551234567`)

## Step 1: Get your API key

1. Sign in to the [Bland dashboard](https://app.bland.ai).
2. Open **Settings** and copy your API key.
3. Store it as an environment variable so you don't commit it to source:

```bash
export BLAND_API_KEY="your-api-key-here"
```

## Step 2: Send a call

Use the `POST /v1/calls` endpoint to dispatch a call. The `task` field describes what the AI agent should do on the call.

<CodeGroup>

```bash cURL
curl -X POST https://api.bland.ai/v1/calls \
  -H "Authorization: $BLAND_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "phone_number": "+15551234567",
    "task": "You are calling to confirm a dentist appointment for tomorrow at 2pm. Be polite and friendly.",
    "voice": "maya",
    "record": true
  }'
```

```javascript Node.js
const response = await fetch("https://api.bland.ai/v1/calls", {
  method: "POST",
  headers: {
    Authorization: process.env.BLAND_API_KEY,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    phone_number: "+15551234567",
    task: "You are calling to confirm a dentist appointment for tomorrow at 2pm. Be polite and friendly.",
    voice: "maya",
    record: true,
  }),
});

const data = await response.json();
console.log(data);
```

```python Python
import os
import requests

response = requests.post(
    "https://api.bland.ai/v1/calls",
    headers={"Authorization": os.environ["BLAND_API_KEY"]},
    json={
        "phone_number": "+15551234567",
        "task": "You are calling to confirm a dentist appointment for tomorrow at 2pm. Be polite and friendly.",
        "voice": "maya",
        "record": True,
    },
)

print(response.json())
```

</CodeGroup>

A successful response returns a `call_id` you can use to track the call:

```json
{
  "status": "success",
  "call_id": "9d404c1b-6a23-4426-953a-a52c392ff8f1"
}
```

## Step 3: Check the call status

Use the `call_id` to fetch call details, including the transcript and recording URL once the call completes.

```bash
curl https://api.bland.ai/v1/calls/$CALL_ID \
  -H "Authorization: $BLAND_API_KEY"
```

You can also watch calls in real time on the [Call Logs](https://app.bland.ai/dashboard/call-logs) page.

## Common parameters

| Parameter | Description |
| --- | --- |
| `phone_number` | The number to call, in E.164 format (e.g. `+15551234567`). |
| `task` | Plain-language instructions for the agent. |
| `voice` | Voice persona for the agent. Browse options in the dashboard. |
| `first_sentence` | The first line the agent speaks. |
| `wait_for_greeting` | If `true`, the agent waits for the recipient to speak first. |
| `max_duration` | Maximum call length in minutes. |
| `record` | Set to `true` to record the call. |
| `webhook` | URL to receive a post-call payload with transcript and metadata. |

See the [Send Call API reference](/api-v1/post/calls) for the full list.

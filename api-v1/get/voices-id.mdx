---
title: "Retrieve Voice Preset" 
api: "GET https://api.bland.ai/v1/voices/{id}"
description: "Retrieves detailed information about a specific voice preset for the given id."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField query="id" type="string" required>
  The unique identifier for the voice preset.

  Used within your call like `voice: "ff2c405b-3dba-41e0-9261-bc8ee3f91f46"` instead of `voice_id: ..., reduce_latency: ...`.
</ParamField>

### Response

<ResponseField type="object" name="voice">
    Contains detailed information about the specified voice preset. 
    
    - `id` - The unique identifier for the voice preset.
    - `voice_name` - Public voice name.
    - `description` - The description of the voice preset.
    - `reduce_latency` - Whether or not the voice preset is optimized for low latency.
    - `interruption_threshold` - The threshold for the voice preset to be interrupted by a new message.
    - `language` - The language code of the voice preset.
    - `voice_settings` - The voice settings of the voice preset.
    - `editable` - Whether or not your account can edit the preset.
    - `public` - Whether or not the preset is publicly available.
    - `created_at` - The date and time the preset was originally created.
</ResponseField>

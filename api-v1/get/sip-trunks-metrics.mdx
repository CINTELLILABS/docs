---
title: "Get SIP Trunk Metrics"
api: "GET https://api.bland.ai/v1/sip/trunks/{trunk_id}/metrics"
description: "Return 30-day call metrics aggregated across every number bound to a trunk."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField path="trunk_id" type="string" required>
  The trunk identifier.
</ParamField>

### Response

<ResponseField name="total_calls" type="number">
  Total call attempts in the last 30 days across all bound numbers.
</ResponseField>

<ResponseField name="answered_calls" type="number">
  Calls that were answered.
</ResponseField>

<ResponseField name="failed_calls" type="number">
  Calls that failed to connect.
</ResponseField>

<ResponseField name="top_failure_codes" type="array">
  The most common SIP failure codes. Each entry includes `sip_code` (number), `description` (string), and `count` (number).
</ResponseField>

```json Example Request
curl -X GET https://api.bland.ai/v1/sip/trunks/trk_abc123/metrics \
  -H "Authorization: Bearer <token>"
```

<ResponseExample>

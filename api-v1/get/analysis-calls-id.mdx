---
title: "Retrieve Call Analysis" 
api: "GET https://api.bland.ai/v1/analysis/calls/{call_id}"
description: "Fetches the AI analysis results for a specific call."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField query="call_id" type="string" required>
    The ID of the call to retrieve analysis for.
</ParamField>

### Response

<ResponseField type="string" name="status">
  Whether the request was successful or not. Possible values are `success` and `error`.
</ResponseField>

<ResponseField type="string" name="message">
  An error message or confirmation that the request was successful.
</ResponseField>

<ResponseField type="array" name="analysis">
  An array of analysis objects, each corresponding to a call within the batch. 
  
  Each analysis object includes:
    - `call_id`, 
    - `goal`
    - `answers` - the results of the AI analysis
    - `questions` - the original questions asked by the AI
</ResponseField>

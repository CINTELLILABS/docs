---
title: "Generate SIP Password"
api: "GET https://api.bland.ai/v1/sip/generate-password"
description: "Generate a cryptographically random SIP password that is safe to drop into PBX config files and SIP URIs."
---

Use this when configuring `register` auth on a trunk or number and you want a strong, URI-safe password without picking one yourself. The generated password avoids characters that need escaping in SIP URIs and PBX config files.

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Query Parameters

<ParamField query="length" type="number">
  Desired password length. Must be between `8` and `256`. Defaults to a strong length when omitted.
</ParamField>

<ParamField query="preset" type="string">
  Convenience preset that sets the length for you. One of `"standard"` or `"long"`. If both `length` and `preset` are provided, `length` wins.
</ParamField>

### Response

<ResponseField name="password" type="string">
  The generated password.
</ResponseField>

<ResponseField name="length" type="number">
  The actual length of the returned password.
</ResponseField>

```json Example Request
curl -X GET 'https://api.bland.ai/v1/sip/generate-password?preset=long' \
  -H "Authorization: Bearer <token>"
```

<ResponseExample>
```json Response
{
  "data": {
    "password": "h4Jq2vNk9LpRx7Tm3Zc8wYbF6sUaQeXg",
    "length": 32
```

</ResponseExample>

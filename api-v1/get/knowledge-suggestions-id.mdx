---
title: "Get Q&A Pair"
api: "GET https://api.bland.ai/v1/knowledge/suggestions/{suggestion_id}"
description: "Returns a single Q&A pair (suggestion) by ID."
---

Retrieves the full details of a Q&A pair.

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField path="suggestion_id" type="string" required>
  The Q&A pair ID (UUID).
</ParamField>

### Response

<ResponseField name="data" type="object">
  The Q&A pair.

  <ResponseField name="data.id" type="string">
    Unique identifier.
  </ResponseField>

  <ResponseField name="data.knowledge_base_id" type="string">
    Knowledge base this pair belongs to.
  </ResponseField>

  <ResponseField name="data.question" type="string">
    The question.
  </ResponseField>

  <ResponseField name="data.answer" type="string">
    The answer.
  </ResponseField>

  <ResponseField name="data.created_at" type="string">
    ISO timestamp of creation.
  </ResponseField>

  <ResponseField name="data.updated_at" type="string">
    ISO timestamp of last update.
  </ResponseField>
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://api.bland.ai/v1/knowledge/suggestions/e5f6a7b8-1234-5678-9abc-def012345678" \
  -H "authorization: YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Error Response
{
  "data": null,
  "errors": [
    {
      "error": "KB_ERROR",
      "message": "Suggestion not found"
    }
  ]
}
```

</ResponseExample>

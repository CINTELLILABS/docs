---
title: "Audio Recording"
api: "GET https://api.bland.ai/v1/calls/:call_id/recording"
description: "Retrieves the specified call's audio recording, if it exists."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField path="call_id" type="string">
  The ID of the call to retrieve the recording for.
</ParamField>

### Response

<ResponseField name="status" type="string">
  Can be `success` or `error`.
</ResponseField>

<ResponseField name="message" type="string">
  If the status is `success`, the `url` field will be present.
  Alternatively if the status is `error`, the message will say "Error fetching recording" or "Internal server error".
</ResponseField>

<ResponseField name="audio" type="file">
  If successful, the audio file will be returned.
</ResponseField>


<ResponseExample>

```json Success response
(Generated Audio File)
```

```json Error response
{
  "status": "error",
  "message": "Error fetching recording"
}
```

</ResponseExample>

---
title: "Leave Organization"  
api: "DELETE https://api.bland.ai/v1/orgs/self/leave"  
description: "Remove the authenticated user from an organization."  
---

### Headers

<ParamField header="authorization" type="string" required>  
  Your API key for authentication.  
</ParamField>  

### Body

<ParamField body="org_id" type="string" required>  
  The unique identifier of the organization the user wants to leave.  
</ParamField>  

### Response

<ResponseField name="data" type="null">  
  Always `null` upon successful removal.  
</ResponseField>  

<ResponseField name="errors" type="null">  
  Always `null` on success.  
</ResponseField>

  "data": null,
  "errors": null
}

</ResponseExample>

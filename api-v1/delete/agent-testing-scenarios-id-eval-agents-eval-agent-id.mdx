---
title: "Detach Eval Agent from Scenario"
api: "DELETE https://api.bland.ai/v1/agent-testing/scenarios/{id}/eval-agents/{evalAgentId}"
description: "Detach an eval agent from a test scenario. Future simulation runs of the scenario will no longer be scored by this eval agent."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField path="id" type="string" required>
  The scenario ID.
</ParamField>

<ParamField path="evalAgentId" type="string" required>
  The eval agent ID to detach.
</ParamField>

### Response

<ResponseField name="detached" type="boolean">
  Whether the eval agent was successfully detached.
</ResponseField>

<ResponseExample>
```json Response
{
  "detached": true
}
```
</ResponseExample>

---
title: "Delete Gap"
api: "DELETE https://api.bland.ai/v1/knowledge/gaps/{id}"
description: "Removes a gap from active results. The question will not be re-created if asked again."
---

Soft-deletes a gap. The gap is removed from the active list and search index. If the same question is asked again in future calls, it will not generate a new gap.

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField path="id" type="string" required>
  The gap ID (UUID).
</ParamField>

### Response

<ResponseField name="data" type="object">
  Deletion result.

  <ResponseField name="data.success" type="boolean">
    Whether the operation succeeded.
  </ResponseField>

  <ResponseField name="data.message" type="string">
    Status message.
  </ResponseField>
</ResponseField>

<RequestExample>

```bash cURL
curl -X DELETE "https://api.bland.ai/v1/knowledge/gaps/14574aa4-892c-4bc7-ac8e-9c17c886aa6d" \
  -H "authorization: YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Error Response
{
  "data": null,
  "errors": [
    {
      "error": "KB_GAPS_ERROR",
      "message": "Gap is already deleted"
    }
  ]
}
```

</ResponseExample>

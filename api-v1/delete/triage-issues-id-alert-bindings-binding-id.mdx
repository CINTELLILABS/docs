---
title: "Unbind Alert"
api: "DELETE https://api.bland.ai/v1/triage/issues/{id}/alert-bindings/{binding_id}"
description: "Unbind an alert from an issue."
---

## Overview

Removes the link between an alert and an issue. The underlying alert is unaffected.

---

## Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

---

## Path Parameters

<ParamField path="id" type="string" required>
  Internal UUID of the issue.
</ParamField>

<ParamField path="binding_id" type="string" required>
  Internal UUID of the binding (the `id` field on a binding object).
</ParamField>

---

## Response

Returns `204 No Content` on success.

Returns 404 if the issue does not exist or the binding does not belong to this issue.


<ResponseExample>
```http No Content
HTTP/1.1 204 No Content
```

```json Not Found
{
  "data": null,
  "errors": [
    { "error": "not_found", "message": "Alert binding not found" }
  ]
}
```
</ResponseExample>

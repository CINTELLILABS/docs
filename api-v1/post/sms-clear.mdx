---
title: "Clear SMS Conversation"
api: "POST https://api.bland.ai/v1/sms/clear"
description: "Hides all messages in an SMS conversation from the AI's future responses (without permanently deleting them). For development/debugging."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>
### Body


<ParamField body="to" type="string" required>
    One of the phone numbers in the conversation.
</ParamField>

<ParamField body="from" type="string" required>
    The other phone number in the conversation.
</ParamField>

<ResponseExample>

```json Response
{
    "status": "success",
    "count": 42 // the number of messages cleared
}
```

</ResponseExample>

---
title: "Update SMS Number"
api: "POST https://api.bland.ai/v1/sms/update"
description: "Update your SMS agent's configuration."
---

<Info>
  **Enterprise Feature** - SMS is only available on Enterprise plans. Contact your Bland representative for access.
</Info>

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>
### Body


<ParamField body="phone_number" type="string">
The phone number to update.
</ParamField>

<ParamField body="prompt" type="string">
The prompt for the AI to use when replying. 
</ParamField>

<ParamField body="ignore_keywords" type="array">
Pass in an array of strings, that if present, the AI should not respond to. Set to `null` to disable.
</ParamField>

<ParamField body="temperature" type="number">
The temperature for prompt and underlying model.
</ParamField>

<ResponseExample>

```json Response
{
  "status": "Prompt updated"
}
```

</ResponseExample>

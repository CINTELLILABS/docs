---
title: "Clone Voice Preset"  
api: "POST https://api.bland.ai/v1/voices/{id}/clone"
description: "Creates a duplicate of a specific voice preset."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField query="id" type="string" required>
  The unique identifier for the voice preset to be cloned.

  Note:
  - Public presets can be cloned by any user as private presets.
  - `' (copy)'` will be appended to the name and description of the cloned preset.
</ParamField>

### Response

<ResponseField type="string" name="message">
    Confirmation message indicating the successful copying of the voice preset.
</ResponseField>

<ResponseField type="object" name="voice">
    Contains detailed information about the cloned voice preset. 
    
    - `id` - The unique identifier for the new voice preset.
    - `voice_name` - Name of the cloned voice preset.
    - `description` - Description of the cloned voice preset.
    - `reduce_latency` - Whether or not the cloned preset is optimized for low latency.
    - `interruption_threshold` - The interruption threshold setting of the cloned preset.
    - `language` - The language code of the cloned preset.
    - `voice_settings` - The voice settings of the cloned preset.
    - `created_by` - The account ID of the user who created the clone.
    - `editable` - Whether or not the clone is editable.
    - `public` - Whether or not the clone is publicly available.
    - `created_at` - The date and time when the clone was created.
</ResponseField>

---
title: "Update Folder"
api: "PATCH https://us.api.bland.ai/v1/pathway/folders/{folder_id}"
description: "Updates the name of a specific folder for the authenticated user."
---


### Headers
<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField path="folder_id" type="string" required>
  The ID of the folder to update.
</ParamField>

### Body Parameters
<ParamField body="name" type="string" required>
  The new name for the folder.
</ParamField>

### Response
<ResponseField name="folder_id" type="string">
  The unique identifier of the updated folder.
</ResponseField>
<ResponseField name="name" type="string">
  The updated name of the folder.
</ResponseField>
<ResponseField name="parent_folder_id" type="string">
  The ID of the parent folder, if applicable.
</ResponseField>

<ResponseExample>
```json Response
{
  "folder_id": "updated_folder_123",
  "name": "Updated Folder Name",
  "parent_folder_id": null
}
```
</ResponseExample>

---
title: "Reopen Gap"
api: "POST https://api.bland.ai/v1/knowledge/gaps/{id}/reopen"
description: "Reopens a previously dismissed gap."
---

Reopens a dismissed gap and resets the `asked_after_dismissal` counter to zero.

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField path="id" type="string" required>
  The gap ID (UUID). Must have status `DISMISSED`.
</ParamField>

### Response

The updated gap object with status `OPEN`.

<RequestExample>

```bash cURL
curl -X POST "https://api.bland.ai/v1/knowledge/gaps/14574aa4-892c-4bc7-ac8e-9c17c886aa6d/reopen" \
  -H "authorization: YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Error Response
{
  "data": null,
  "errors": [
    {
      "error": "KB_GAPS_ERROR",
      "message": "Can only reopen gaps with DISMISSED status"
    }
  ]
}
```

</ResponseExample>

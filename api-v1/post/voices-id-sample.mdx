---
title: "Generate Voice Sample"  
api: "POST https://api.bland.ai/v1/voices/{id}/sample"
description: "Generates an audio sample from a specified voice preset, with options to override default settings for tuning and testing."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField query="id" type="string" required>
  The unique identifier for the voice preset to be used for generating the sample.
</ParamField>

### Request Body

<ParamField body="text" type="string" required>
  The text content to be spoken in the voice sample. 

  Character limit: `200` characters.
</ParamField>

<ParamField body="voice_settings" type="object" optional>
  Alternate `voice_settings` can be passed in to override the preset's default settings.
</ParamField>

### Response

<ResponseField type="audio" name="sample">
    The generated audio file of the spoken text using the specified or overridden voice preset settings.
</ResponseField>

<ResponseExample>
```json
(Generated audio file)
```
</ResponseExample>

---
title: "Clone a Voice" 
api: "POST https://api.bland.ai/v1/voices"
description: "Clone a voice from audio samples."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Body

<ParamField body="name" type="string" required>
  The name to identify your cloned voice.
</ParamField>

<ParamField body="audio_samples" type="file[]" required>
  One or more audio files to use as samples for voice cloning.
  Must be sent as form-data.
</ParamField>

### Response

<ResponseField name="voice_id" type="integer">
  The unique identifier for the cloned voice.
</ResponseField>

<ResponseField name="name" type="string">
  The name you provided for the voice.
</ResponseField>

<ResponseExample>
```json
{
  "voice_id": 12345,
  "name": "My Custom Voice"
}
```
</ResponseExample>

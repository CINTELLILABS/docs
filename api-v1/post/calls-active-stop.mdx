---
title: "Stop All Active Calls"  
api: "POST https://us.api.bland.ai/v1/calls/active/stop"
description: "End all active phone calls on your account."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Response

<ResponseField name="status" type="string">
  Can be `success` or `error`.
</ResponseField>

<ResponseField name="message" type="string">
  If the status is `success`, the message will say "Call ended successfully."
  Otherwise, if the status is `error`, the message will say "SID not found for
  the given c_id." or "Internal server error."
</ResponseField>

<ResponseField name="num_calls" type="integer">
    The number of active calls that will be cancelled.
</ResponseField>


<ResponseExample>
```json
{
  "status": "success",
  "message": "Stopping active calls. This may take some time...",
  "num_calls": 12
}
```
</ResponseExample>

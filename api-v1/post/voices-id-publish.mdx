---
title: "Publish Voice Preset" 
api: "POST https://api.bland.ai/v1/voices/{id}/publish"
description: "Publishes a specific voice preset, making it publicly available and uneditable."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField query="id" type="string" required>
  The unique identifier for the voice preset to be published.
</ParamField>

### Response

<ResponseField type="string" name="message">
    Confirmation message indicating the successful publication of the voice preset.
</ResponseField>

<ResponseField type="object" name="voice">
    Contains detailed information about the published voice preset. 
    
    - `id` - The unique identifier for the new, published voice preset.
    - `voice_name` - Name of the published voice preset.
    - `description` - Description of the published voice preset.
    - `reduce_latency` - Whether or not the published preset is optimized for low latency.
    - `interruption_threshold` - The interruption threshold setting of the published preset.
    - `language` - The language code of the published preset.
    - `voice_settings` - The voice settings of the published preset.
    - `editable` - Indicates that the preset is no longer editable post-publishing.
    - `public` - Indicates that the preset is now publicly available.
    - `created_at` - The date and time when the preset was published.
</ResponseField>

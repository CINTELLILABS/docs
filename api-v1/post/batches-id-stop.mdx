---
title: "Stop Batch"
api: "POST https://api.bland.ai/v2/batches/{batch_id}/stop"
description: "Stop a batch that is currently running or scheduled."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField path="batch_id" type="string" required>
  The unique identifier of the batch to stop.
</ParamField>

### Response

<ResponseField name="data" type="null">
  Always `null` on success.
</ResponseField>

<ResponseField name="errors" type="null">
  `null` if the batch was successfully stopped.
</ResponseField>

<ResponseExample>

```json Response
{
  "data": null,
  "errors": null
}
```
</ResponseExample>

---
title: "Delete Voice Preset"  
api: "POST https://api.bland.ai/v1/voices/{voice_preset_id}/delete"
description: "Deletes a voice preset."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField path="voice_preset_id" type="string" required>
  The unique identifier for the voice preset to be deleted.

  Note:
  - This voice preset must have been created with your account
  - Default or other user's voice presets cannot be deleted.
  - Public voice presets can be deleted, but not modified.
  - This will not delete cloned voice presets, only the original.
</ParamField>

### Response

<ResponseField type="string" name="message">
    Confirmation message indicating successful deletion of the voice preset.
</ResponseField>

<ResponseField type="string" name="voice">
    The ID of the voice preset that was deleted.
</ResponseField>

<ResponseExample>
```json
{
    "message": "Successfully deleted voice preset with an id of 79698ec7-35c5-44ff-8ad8-2f63ec6104d3",
    "voice": "79698ec7-35c5-44ff-8ad8-2f63ec6104d3"
}
```
</ResponseExample>

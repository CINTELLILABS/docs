---
title: "Rename a Voice" 
api: "PATCH https://api.bland.ai/v1/voices/{id}/rename"
description: "Update the name of an existing cloned voice."
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Path Parameters

<ParamField path="id" type="string" required>
    The `id` of the voice to update.
</ParamField>


### Body

<ParamField body="name" type="string" required>
  The new name for your cloned voice.
</ParamField>

### Response

<ResponseField name="voice_id" type="integer">
  The unique identifier of the renamed voice.
</ResponseField>

<ResponseField name="name" type="string">
  The updated name of the voice.
</ResponseField>

<ResponseExample>
```json
{
  "voice_id": 12345,
  "name": "Updated Voice Name"
}
```
</ResponseExample>

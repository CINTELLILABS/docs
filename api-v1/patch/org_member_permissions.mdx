---
title: "Update Organization Member Permissions"  
api: "PATCH https://api.bland.ai/v1/orgs/{org_id}/members/permissions"  
description: "Modify the permissions of an existing member within an organization."  
---

### Headers

<ParamField header="authorization" type="string" required>  
  Your API key for authentication.  
</ParamField>  

### Path Parameters

<ParamField path="org_id" type="string" required>  
  The unique identifier of the organization.  
</ParamField>  

### Body

<ParamField body="action" type="string" required>  
  The action to perform on the member's permissions.  
  <br/>Valid values: `"add"`, `"remove"`, `"reset"`, `"set"`  
</ParamField>  

<ParamField body="target" type="string" required>  
  The user ID of the member whose permissions are being modified.  
</ParamField>  

<ParamField body="permissions" type="string[]" required>  
  A list of permissions to update.  
  <br/>Valid values: `"owner"`, `"admin"`, `"operator"`, `"viewer"`  
</ParamField>  

### Response

<ResponseField name="data" type="object">  
  Contains the updated permissions for the user.  
</ResponseField>  

<ResponseField name="data.newPermissions" type="string[]">  
  The updated list of permissions assigned to the user.  
</ResponseField>  

<ResponseField name="errors" type="null">  
  Always `null` on success.  
</ResponseField>  

<ResponseExample>  

```json Response  
{
  "data": {
    "newPermissions": ["viewer", "operator"]
  },
  "errors": null
}
```
</ResponseExample>

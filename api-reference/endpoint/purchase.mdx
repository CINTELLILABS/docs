---
title: "Purchase inbound number"
api: "POST https://api.bland.ai/purchasenumber"
description: "Purchase and configure a new inbound phone number"
---

### Headers

<ParamField header="authorization" type="string">
  Your API key for authentication.
</ParamField>
### Body

<ParamField body="area_code" type="string">
  Defaults to a randomly chose toll-free number. To purchase a number with a
  specific area code, set this parameter to the desired three-digit area code. -
  Toll-Free Numbers: Leave blank, "", or default. - Local Number Examples:
  "212", "415", "563".
</ParamField>

<ParamField body="prompt" type="string">
  This defines how the AI will start the conversation, information available to
  it, and its behaviors. Note: you should continually test and iterate prompts
  to get the best output from your AI.
</ParamField>

<ParamField body="webhook" type="string">
  The webhook should be a http / https callback url. We will send the call_id
  and transcript to this URL after the call completes. This can be useful if you
  want to have real time notifications when calls finish.
</ParamField>

### Response

<ResponseField name="phone_number" type="string">
  The created phone number, will be in the following format: +1XXXXXXXXXX -
  Example: +18582814611
</ResponseField>

<ResponseField name="prorated_price" type="float">
  The prorated price of the phone number for the remainder of the month. Note:
  Measured in credit-minutes. - Example: Halfway through the month, the prorated
  price would be half of the recurring_price.
</ResponseField>

<ResponseField name="recurring_price" type="float">
  The recurring price of the phone number per month. Note: Measured in
  credit-minutes. - Example: 16 credit-minutes per month.
</ResponseField>

<ResponseField name="remaining_credits" type="float">
  The remaining credits in your account. Note: Measured in credit-minutes. -
  Example: 92.8017849425806 credit-minutes.
</ResponseField>

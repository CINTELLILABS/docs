---
title: "Wait on Hold (HoldForMe)"
api: "POST https://api.bland.ai/hold"
description: "This endpoint sends an outbound call where the AI navigates customer service and waits on hold until a human being answers. Then it dispatches a call to a number you provide, patching in another human."
---

### Body

<ParamField body="phone_number" type="string">
  The phone number to call (the number of the insurance company, airline, bank,
  etc).
</ParamField>

<ParamField body="hold_connect" type="string">
  The number to connect into the call when a human is detected (your number).
</ParamField>

<ParamField body="voice_id" type="number">
  This is the ID of the AI voice you'd like to use. By default we offer voices 1
  (Kylie), 2 (Vanessa), and 3 (Aditya).
</ParamField>

### Response

<ResponseField name="status" type="string">
  Can be "success" or "error".
</ResponseField>

<ResponseField name="message" type="string">
  A description or reason for the response status.
</ResponseField>

<RequestExample>

```javascript Example Request.js
// Headers
const headers = {
  Authorization: "sk-xxxxxxxxx",
};

// Data
const data = {
  phone_number: "xxx-xxx-xxxxx",
  hold_connect: "xxx-xxx-xxxxx",
  voice_id: 2,
};

// API request
await axios.post("https://api.bland.ai/hold", data, { headers });
```

</RequestExample>

<ResponseExample>

```json Response
{
  "status": "success",
  "message": "The call was successfully put on hold."
}
```

</ResponseExample>

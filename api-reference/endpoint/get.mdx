### Body

<ParamField body="phone_number" type="string">
  This is the phone number of the person or company you want to call
</ParamField>

<ParamField body="voice_id" type="number">
  This is the ID of the AI voice you'd like to use. By default we offer voices 1
  (Kylie), 2 (Vanessa), and 3 (Aditya).
</ParamField>

<ParamField body="request_data" type="object">
  Provides additional context for the AI to reference during the call.
</ParamField>

<ParamField body="reduce_latency" type="boolean">
  Reducing latency can improve response time but may lead to issues like failed
  calls and voice artifacts (issues with rendering voice).
</ParamField>

<ParamField body="webhook" type="string">
  A callback url for after the call. Sends a POST request to a URL when the call
  is finished. Will contain call id, and the transcripts.
</ParamField>

<ParamField body="amd" type="boolean">
  Enables Automatic Machine Detection to detect if the system is speaking with a
  robot. Useful for detecting voicemails or navigating phone trees. Should be
  set to false if you're sure you're calling a human or need quicker response
  time for the first reply.
</ParamField>

### Response

<ResponseField name="status" type="string">
  Either returns 'success' or 'error'
</ResponseField>

<ResponseField name="call_id" type="string">
  A unique identifier for the call (present only if status is 'success')
</ResponseField>

<RequestExample>

```javascript Example Request.js
// Headers
const headers = {
  Authorization: "sk-xxxxxxxxx",
};

// Data
const data = {
  phone_number: "xxx-xxx-xxxxx",
  voice_id: 2,
  task: "You are calling to see if United Airlines has your bags",
  request_data: {
    calling: "United",
    bag_claim: "69683",
    airline_code: "UA123", // Additional context based on requirements
  },
  reduce_latency: false,
  amd: true, // Set to false if you're sure you're calling a human or need quicker response
  webhook: "your-url-here",
};

// API request
await axios.post("https://api.bland.ai/call", data, { headers });
```

</RequestExample>

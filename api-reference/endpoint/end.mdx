---
title: "End phone call"
api: "POST https://api.bland.ai/end"
description: "Send a POST request to end a phone call that's in progress"
---

### Body

<ParamField body="call_id" type="string">
  The unique identifier for the call.
</ParamField>

### Response

<ResponseField name="status" type="string">
  Either 'success' or 'error'
</ResponseField>

<ResponseField name="message" type="string">
  If the status is 'success', the message will say 'Call ended successfully.'
  Otherwise if the status is 'error', the message will say 'SID not found for
  the given c_id.' or 'Internal server error.'
</ResponseField>

<RequestExample>

```javascript Example Request.js
// Headers
const headers = {
  Authorization: "sk-xxxxxxxxx",
};

// Data
const data = {
  call_id: "xxx-xxx-xxxxx",
};

// API request
await axios.post("https://api.bland.ai/end", data, { headers });
```

</RequestExample>

<ResponseExample>

```json Response
{
  "status": "success",
  "message": "Call ended successfully."
}
```

</ResponseExample>

title: "Get transcript"
api: "GET https://api.bland.ai/logs"
description: "Note: A WebSocket stream is also available in beta, which can be more efficient for receiving real-time updates. You can configure it within the developer portal."
---

### Body

<ParamField body="call_id" type="string">
  The unique identifier for the call.
</ParamField>

### Response

<ResponseField name="c_id" type="number">
  The unique identifier for the call.
</ResponseField>

<ResponseField name="to" type="string">
  The phone number that received the call.
</ResponseField>

<ResponseField name="request_data" type="string">
  JSON string representing additional data about the call.
</ResponseField>

<ResponseField name="completed" type="boolean">
  Whether the call has been completed.
</ResponseField>

<ResponseField name="created_at" type="string">
  The timestamp when the call was created.
</ResponseField>

<ResponseField name="transcripts" type="array of objects">
  An array of spoken phrases during the call.
</ResponseField>

<ResponseField name="call_length" type="number or null">
  The length of the call in seconds (or null if ongoing).
</ResponseField>

```javascript Example Request.js
// Headers
const headers = {

};

// Data

```

## Sending your first AI phone call

Here's the code snippet.

<CodeGroup>
```javascript SendCall.js
// Headers
const headers = {
  Authorization:
    "sk-a6p90s4w6k69w2ak1hanl40fuetoso5fgd1pmnt87gpsepl9dsw30stfpx7x95r469",
};

// Data
const data = {
phone_number: "(111) 222-3344",
task: "Welcome the developer to Bland AI!",
voice_id: 1,
reduce_latency: true,
request_data: {},
};

// API request
await axios.post("https://api.bland.ai/call", data, { headers });

````

```javascript SendCall.py
# Headers
headers = {
  'Authorization': "sk-a6p90s4w6k69w2ak1hanl40fuetoso5fgd1pmnt87gpsepl9dsw30stfpx7x95r469"
}

# Data
data = {
  'phone_number': '(111) 222-3344',
  'task': 'Welcome the developer to Bland AI!',
  'voice_id': 1,
  'reduce_latency': true,
  'request_data': {}
}

# API request
requests.post('https://api.bland.ai/call', json=data, headers=headers)
````

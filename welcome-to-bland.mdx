Bland is an AI phone calling platform for building, deploying, and managing AI voice agents at scale. Send outbound calls, configure inbound numbers, build conversational pathways, and integrate external tools through three first-class developer surfaces: a [CLI](/sdks/cli), an [AI-powered Dev Terminal](/sdks/dev-terminal) with bundled MCP tools, and a [direct API](/api-v1/post/calls). The same task prompt powers every path. Production deployments run thousands of regulated calls per day with sub-400ms response latency and SOC 2 Type II, HIPAA, GDPR, and PCI DSS compliance. Need help? The Bland team runs daily office hours and live support on [Discord](https://discord.gg/QvxDz8zcKe).

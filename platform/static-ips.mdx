## Overview

Bland sends outbound HTTP requests from the source IPs below, based on your organization's residency.

These IPs apply to webhooks, custom API requests, and similar HTTP traffic. SIP signaling and media use separate IPs; see [SIP Connection Endpoints](/enterprise-features/SIP-integration#connection-endpoints) or [`GET /v1/sip/firewall-ips`](/api-v1/get/sip-firewall-ips).
